Tips

Select at least one table from the list. Remember that selecting one table will compare it to the internal database table. Currently it can only compare two tables at once.


The current supported formats are files with comma separated values, like .csv or .txt. In doubt stick to the .csv format as the others are not guaranteed to work.


Keep the formatting of the columns in the search field separated by commas, any other punctuation or unnecessary characters will not work. Ex: column_name_1, column_name_2, column_name_3 and so on.


Keep the columns names in matching order, the first column name of the first field will match the first column name of the second field and so on and so forth. If you write four names in the first field you also need to write four names in the second field.


The more rows you can match between the tables the better will be the result!
